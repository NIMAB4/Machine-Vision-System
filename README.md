## Quick Example

Below is a short script demonstrating how to load an image with OpenCV, convert it to grayscale, and display it using Matplotlib:

```python
import cv2
import matplotlib.pyplot as plt

# Load an image (replace with your own image path)
img = cv2.imread('sample.jpg')
# Convert BGR (OpenCV default) to RGB for Matplotlib
img_rgb = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)
# Convert to grayscale
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# Plot original and grayscale side by side
fig, axs = plt.subplots(1, 2, figsize=(10, 5))
axs[0].imshow(img_rgb)
axs[0].set_title('Original (RGB)')
axs[0].axis('off')
axs[1].imshow(gray, cmap='gray')
axs[1].set_title('Grayscale')
axs[1].axis('off')
plt.show()
```

Save the script as `example.py` and run it:
```bash
python example.py
```

You should see a window displaying the original image and its grayscale version.
